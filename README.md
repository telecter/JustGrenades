# JustGrenades

A tiny, but explosive, Minecraft mod.

This mod adds a few types of craftable grenades to Minecraft.

### Normal Grenade
Creates an sizeable explosion wherever it lands.
![recipe](recipe.png)

### Smoke Grenade
Creates a cloud of smoke and temporarily blinds anyone in the vicinity of wherever it lands.
![recipe](recipe_smoke.png)
